Privacy Policy for LeetCode Zen Mode

Effective date: July 2, 2026

LeetCode Zen Mode is a browser extension designed to hide selected interface elements on LeetCode so users can focus on solving problems.

Information We Collect

LeetCode Zen Mode does not collect, transmit, sell, or share personal data.

Settings Stored by the Extension

The extension stores user preferences such as:

  • whether Zen Mode is enabled
  • which LeetCode elements the user wants to hide
  • separate settings for contest pages, if configured

These preferences are stored using Chrome extension storage so the extension can remember the user's settings.

Data We Do Not Collect

LeetCode Zen Mode does not collect:

  • personally identifiable information
  • health or financial information
  • passwords or authentication credentials
  • personal communications
  • location data
  • web browsing history outside the extension's purpose
  • keystrokes, mouse activity, or analytics
  • website content for transmission or storage on external servers

Third-Party Services

LeetCode Zen Mode does not send user data to external servers or third-party analytics services.

Changes to This Policy

If this privacy policy changes in the future, the updated version will be published at this page.

Contact
